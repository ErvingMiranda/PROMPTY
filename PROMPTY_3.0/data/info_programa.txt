SOBRE EL PROGRAMA
PROMPTY 3.0 es un asistente virtual de escritorio desarrollado como proyecto final para el curso "Introducción a la Programación" en la Universidad Americana (UAM).

Objetivo del programa:
Desarrollar un asistente virtual en Python capaz de ejecutar tareas básicas mediante comandos por texto o voz, integrando una voz sintética natural, autenticación de usuarios, y una estructura modular profesional orientada a objetos. Esta versión está diseñada para permitir su futura integración con una interfaz gráfica.

PROMPTY busca ofrecer una experiencia funcional y cercana a la de asistentes reales, permitiendo a los usuarios realizar acciones cotidianas como consultar la hora, abrir archivos, buscar en internet o recibir datos curiosos, todo con una estructura flexible y moderna.

Funciones actuales:
- Mostrar fecha y hora actual.
- Abrir archivos o carpetas por ruta.
- Realizar búsquedas en YouTube.
- Compartir datos curiosos desde una lista interna.
- Proporcionar información sobre el programa y sus creadores.
- Recibir comandos por texto o voz.
- Responder mediante voz sintética.
- Iniciar sesión con distintos roles y permisos.

PROCESO DE DESARROLLO
PROMPTY está desarrollado en Python 3.13.3 utilizando librerías estándar como `datetime`, `os`, `webbrowser`, `random`, así como módulos como `speech_recognition` y `pyttsx3` para integrar entrada y salida por voz.

A diferencia de versiones anteriores, PROMPTY 3.0 utiliza una estructura profesional basada en capas (data, models, services, views) y programación orientada a objetos. Esto permite que el código sea fácilmente reutilizable en su interfaz gráfica.
